##### Для разработки:

даем команду **npm run dev**

##### Для продакшена:

даем команду **npm run prod**

Получаем папку dist содержащую минимизированный / сжатый код


##### Для проверки правильности кода:

даем команду  **npm run lint**


##### Для проверки тестов:

даем команду  **npm run test**
